# CBJQ_XiaoKaiGuan

尘白禁区 *XiaoKaiGuan*（小开关😎😋）

这个项目很简单，只是为了方便不那么熟悉Windows系统各种操作的玩家直接开关小开关使用。

## 项目历史

前有更加简单的实现，即bat脚本，但无法加上图标，不那么美观。原先在作者的Gist中有一份，叫做[`Enya_Switch`](https://gist.github.com/LiuJiewenTT/ef6966810da80200fce8a3e8ff13e69e)，启动和结束都是单独的脚本，也就是说，如果创建快捷方式到桌面，可能需要两个快捷方式。另外，原脚本需要玩家手动附上管理员权限，不方便。

## 项目介绍

这个项目使用C语言实现，提供开和关的统一管理入口，并展示图标。
